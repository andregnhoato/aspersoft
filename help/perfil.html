Comparação de perfis

O que é e para que serve a comparação de Perfil?

A tela de Comparação de Perfis é utilizada para visualizar via gráfico como ficou o perfil de distribuição dos ensaios e fazer a comparação de vários ensaios.

  • O botão , limpa o gráfico para novas comparações.
